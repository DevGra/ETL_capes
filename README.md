
# ETL_capes_em_R
E.T.L dos dados da CAPES(discentes, docentes, programas, instituições) dos anos de 2013 a 2017.

É um E.T.L foi construído utilizando a linguagem R, utilizando vários pacotes, separados em um arquivo e carregado quando se faz o download dos dados(csv). Ao final, estes dados são separados para serem utilizados nos gráficos dinâmicos através do Shiny, que é um sistema para desenvolvimento de aplicações web usando o R. Ex:
